AR Enhanced Children's Book Concept

This concept was created as a result of my curiosity about augmented reality (AR). Specifically, how AR can be used in the classroom to help children learn better. This concept would teach children about dinosaurs, but this idea can be used with many different subjects.

The idea is that the book will contain pages with information about a species of dinosaur, much like a traditional book. But it would also contain pages with AR codes which can be scanned to reveal 3-dimensional models of that dinosaur. Children would be able to interact with the 3-D model for a more engaging learning experience.

TConcept built using Zappar and Zapworks Studio.
